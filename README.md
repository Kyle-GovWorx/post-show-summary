# Post-Show Summary Report

A single-page web form for capturing post-conference summaries with one-click PDF export.

## Features
- All fields from the original Post-Show Summary template (Conference Details, Show Details, Marketing, Notes, Ratings)
- 1-5 star ratings (click again to clear)
- One-click PDF export, named after the conference
- No backend, no install — open `index.html` in any modern browser

## Use it
- Locally: open `index.html`
- Hosted: enable GitHub Pages on this repo (Settings → Pages → Deploy from main, root)
